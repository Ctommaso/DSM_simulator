
Install Python 2.7 as well as the libraries
-numpy
-matplotlib
-scipy
-pandas
-Tkinter
-PIL


Compile the c++ executables:

- Open a terminal
- navigate into to /src
- Compile using the command lines below

Command lines to compile are: 
	make -f Makefile

Command lines to clean up are: 
	make -f Makefile clean
	

To run the code:
- Open a terminal
- Navigate to the directory containing Sim_AutoQuar.py
- Type python Sim_AutoQuar.py
